Class McsvLoader

Description
Description | Vars | Methods (details)

McsvLoader is a facility to load a CSV file into a mysql table.

A new table name is always sensibly created from the file name.
Label names are analysed to create column names.
Data is analysed to determine column datatypes, most date formats are well understood.
In tandem with Mquery, this can be a useful ad-hoc analysis tool for Excel data.
